What is Web Storage?

localStorage and sessionStorage are part of the Web Storage API, which allows websites to store data in a user's browser.

localStorage stores data with no expiration time. This data remains available even after the browser is closed and reopened.

sessionStorage stores data for the duration of a page session. It is cleared when the page is closed.

These are commonly used to store user preferences, temporary data, or session-based information without needing a server.
